# Toasty
Simple Toast Library.

> Step 1. Add the JitPack resitory to your build file

```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  > Step 2. Add the dependancy
  ```gradle
  	dependencies {
	        implementation 'com.github.lasithadilshan:Toasty:1.0.0'
	}
  ```
